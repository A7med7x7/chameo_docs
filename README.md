# About

This repository contains the source code for the Chameo documentation.

## How to Serve the Website for Development

To serve the Chameo documentation locally for development, follow these steps:

1. **Install MkDocs**  
   Ensure you have Python installed (version 3.9 or higher). 

   Then, install MkDocs and the Material theme by running:

   ```bash
   pip install mkdocs mkdocs-material
   ```

   in the project root directory run:
   
   ```bash
   mkdocs serve
   ```
